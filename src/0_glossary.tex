\section*{PostGISに関する用語}


\begin{itemize}
    \item "POLYLINE" ー POLYLINEとはPostGISがPostgreSQLシステムにもたらす地理情報型であり, 座標と座標を結ぶ線分を記録する.
    \item "POLYGON"  ー POLYGONとはPostGISがPostgreSQLシステムにもたらす地理情報であり, 平面図形の頂点を結ぶ平面図形を意味する.
\end{itemize}
