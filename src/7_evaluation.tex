\chapter{評価}
\label{evaluation}

本章では，提案システムの評価について述べる．


\section{評価内容}

本研究の実験において以下の項目について評価を行う.


\begin{itemize}
    \item DQNモデルによって出力された経路選択はユースケースに挙げた項目を満たせているか.
    \item ユースケース満たした上で, なるべく距離が短いルートを選択できているか.
\end{itemize}
